KKU Student Management System

Project Overview:
This is a comprehensive management system developed in JAVA to handle student registrations, academic grading, and data analysis. The project was designed with a focus on Object-Oriented Programming (OOP) principles.

Technical Features:
OOP Implementation: Utilizes Inheritance, Polymorphism, and Interfaces to manage different student types (Undergrad vs. Grad).
Data Persistence: Implements File I/O (BufferedReader/BufferedWriter) to save and retrieve student records locally in `students_data.txt`.

Technologies Used
Language: Java (JDK 11+)
Concepts: OOP, File Handling, Data Structures
IDE: Eclipse
